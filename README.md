Express JS Example Request and Response using JSON and XML Format

Install package
```bash
npm install
```

Run app 
```bash
node index.js
```


Local: 
```
localhost:9001
```


Endpoint
```bash
GET localhost:9091/rss?source=1&key=XXX

source : Indonesian news portal rss url code (1,2,3,4,5)
key: Search Key
```
